Ocean Data Lab, 2022 Summer

Ocean Data Lab Website: view

My development product: view

Context

In the past, members of the Ocean Data Lab have collected over 1.5GB ocean acoustic data from large underwater networks. Various types of data visualizations are created to better analyze those data. However, Those visualizations requires different inputs such as location, time interval, graph type, etc, to be generated. It will be much convenient to manipulate these required parameters on an interface than writing Python codes.

Responsibility

Created a website where users can select different locations, graph types, and time interval to generate different data visualizations. Website also allows users to download the acoustic data in CSV file based on different requirements.

Takeaway

I chosed a tech combination (React + Flask) that I never tried before. My experience of building the interface highly improves my problem-solving skills and research skills. I got to know Professor Shima and a group of amazing researchers.

Topic
