Reflection - Dev Tool Chain

This is my HCDE 438 reflection, where I reflect on my experience in the class and the projects I worked on.

Overview and Usage

Overview: I am using the p5.js library and have created a gluttonous snake game. To start playing, click the "Start Game" button and you will enter the game immediately. There is a score panel in the top left corner that displays the number of points you have earned (which corresponds to the number of red boxes the snake eats). If the snake eats itself or hits the wall (the edge of the browser window), the game is over. You can restart the game by clicking the "Start Game" button again. Enjoy the game!

Or, you can click the linked text below to view the game page. https://andrewliu66.github.io/snake-game/

Here is the github repository link: https://github.com/AndrewLiu66/snake-game

Development Process
Here's a step-by-step overview of how to build the gluttonous snake game using p5.js:
  1. Set up the basic HTML structure for the game, including a canvas element and a "Start Game" button. Also include a script tag to include the p5.js library.
  2. Define global variables for the snake, food, scale, score, and a flag to indicate whether the game has started.
  3. Create the setup function. In this function, create the canvas, create a new snake object, set the frame rate, and call the pickLocation function to create the initial location of the food.
  4. Create the draw function. In this function, set the background color, check if the game has started, update the position of the snake, check if the snake has eaten the food, check if the snake has died, and draw the snake and food on the canvas.
  5. Create the keyPressed function. In this function, set the direction of the snake based on the key that was pressed.
  6. Create the pickLocation function. In this function, generate a random location for the food.
  7. Create the restartGame function. In this function, reset the score, create a new snake object, and call the pickLocation function to create the initial location of the food. Also reset the gameStarted flag to false.
  8. Define the Snake class. In the constructor function, initialize the head of the snake at the center of the canvas. In the dir function, set the direction of the snake based on the x and y parameters. In the update function, update the position of the head and tail of the snake. In the death function, check if the snake has died by going off the screen or hitting its tail. In the eat function, check if the snake has eaten the food. In the show function, draw the snake on the canvas.
  9. Add an event listener to the "Start Game" button that calls the setup function and sets the gameStarted flag to true.
  10. Test the game and make adjustments as needed.

Issue
During the game creation process, I encountered an issue where the snake would not stop once it hit the wall. I struggled for a bit to find a solution, but here are the steps I took to resolve the issue:
  1. I located the function that defines the end condition of the game.
  2. I modified the condition in the if statement to fix the bug.
  3. I replayed the game to ensure that it worked.
As a result of these steps, I was able to resolve the issue and make the game work.
