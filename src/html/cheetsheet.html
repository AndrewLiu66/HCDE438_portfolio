CSS reference
  • control the timing of the transition using the

    transition-timing-function

    property.
  • specify the speed curve of the transition using the

    transition-timing-function

    property
  • specify a delay before the transition starts using

    transition-delay

    property
  • specify the length of time that the transition should take to complete using the

    transition-duration

    property
  • use the

    none

    keyword to disable all transitions for an element
      

        .element {
          /* disable transitions for this element */
          transition: none;
        }

        .element:hover {
          /* enable transitions for this element */
          transition: all 0.5s ease-in-out 0.5s;
        }
