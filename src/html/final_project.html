Final Project: The Ultimate Snake Game

This is my HCDE 438 reflection, where I reflect on my experience in the class and the projects I worked on.

Overview and Usage

Overview: I am using the p5.js library to create a gluttonous snake game. To start playing, click the "Start Game" button, and you will enter the game immediately. There is a score panel in the top left corner that displays the number of points you have earned (which corresponds to the number of red boxes the snake eats). If the snake eats itself or hits the wall (the edge of the browser window), the game is over. There are power-up foods embedded in this new version of the game. This special food will either speed up the snake or make the snake longer. You can restart the game by clicking the "Start Game" button again. Enjoy the game!

Or, you can click the linked text below to view the game page. https://andrewliu66.github.io/final_snake_game/index.html

Here is the github repository link: https://github.com/AndrewLiu66/final_snake_game

Final Project scope
My final version of the Snake game now includes more interesting features that were specified in the proposal and have been implemented. Building upon the prototype, this version includes an introduction panel that explains the rules of the game and the different types of fruit that can be eaten by the snake, along with the corresponding abilities that the snake gains after eating each fruit. Additionally, I have added a report page that appears after the game ends, allowing users to view their scores. To help users keep track of their highest score, I utilized the browser's local storage to store this information. As a result, my project scope has expanded beyond its initial plan, and these additions have significantly improved the overall quality and user experience of the game.
Final Project Time management Summary
Adding the introduction panel and game-over page to the game was relatively straightforward. However, adding the feature to record the user's highest score took longer. I had to learn how to use the browser's local storage to store the highest score, and figure out where to call the "localStorage.getItem" and "localStorage.setItem" functions in the game. Consequently, I dedicated most of my time to solving this problem.
Development Process
The final version of the game was built on top of the previous prototype, Here's a step-by-step overview of how to add new features to the gluttonous snake game using p5.js:
  1. Identify the new features to be added to the game, which include an introduction page and a game-over page.
  2. Created new HTML pgae to show the "gameover" page
  3. Create a new JavaScript file for the introduction page, and write the necessary code to display the rules of the game.
  4. Create a new JavaScript file for the game-over page, and write the necessary code to display the user's score.
  5. Modify the draw function to include new features such as displaying the score and updating the game-over page.
  6. Update the Snake class or any other class used in the game to accommodate the new features.
  7. Test the game with the new features and make necessary adjustments.
  8. Optimize the code and remove any bugs that may have arisen during the implementation of the new features.
  9. Publish the updated game for users to enjoy.
