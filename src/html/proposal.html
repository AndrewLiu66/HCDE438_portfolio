Project Proposal: Enhanced Snake Game with Power-ups and Obstacles

In this project, I will enhance the classic snake game by adding power-ups and obstacles to make the game more exciting and challenging. The game will be built using the p5.js library.

Drawing
Descriptions of two versions of my project
1.Prototype Version: contains functional core features for the snake game but with minimal CSS styling and without instructions about the new features. Here are the new features available in the prototype version.

****Power-ups****: The game will have power-ups that appear randomly on the screen and give the snake different abilities or bonuses. Some examples of power-ups include:
  • Speed Boost: Increases the snake's speed for a limited time.
  • Longer Body: Adds extra segments to the snake's body, making it longer.
****Obstacles****: The game will have obstacles that the snake needs to avoid to stay alive. Some examples of obstacles include:
  • Walls: The snake will need to avoid hitting the walls to prevent losing the game.
2. Final Version: The game will have all the functional features, including the new power-up and obstacle features with good color schemes. Detailed instructions introducing the new features will be shown to users before they play the game.
Development Process
Here's a step-by-step overview of how to build my updated gluttonous snake game using p5.js:
  1. Define the scope and requirements of the snake game.
  2. Develop the core functionality of the snake game, including the snake movement and collision detection.
  3. Create a prototype version of the game with minimal CSS styling and without instructions about the new features.
  4. Test the prototype version of the game and gather user feedback.
  5. Incorporate user feedback and develop new power-up and obstacle features, such as speed boost and walls.
  6. Implement good color schemes for the power-ups and obstacles.
  7. Develop detailed instructions introducing the new features for the game.
  8. Test the final version of the game and ensure that all functional features are working properly.
  9. Release the final version of the game to the public.

List of unknowns
Here's a list of unknowns:
  1. How to generate power-ups and obstacles randomly on the screen.
  2. How to code the logic for power-ups and obstacles to interact with the snake and affect its behavior.
  3. How to create good color schemes for the power-ups and obstacles.
List of Stretch Goals
  1. Multiplayer mode: Allow multiple players to play the game together online or locally.
  2. Advanced AI opponents: Implement computer-controlled opponents with advanced AI that can challenge the player.
  3. Customizable snake skins: Allow players to customize the appearance of their snake with different skins or patterns.
