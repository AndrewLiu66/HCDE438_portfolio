Issue
I recently encountered an issue when trying to push my changes to a remote repository on GitHub. When I ran the git push command, I received an error message that said "fatal: The current branch master has no upstream branch." After some investigation, I realized that the issue was caused by a misconfigured local repository. To resolve the issue, I followed the steps below:
  1. Check the status of my local repository by running the command git status.
  2. If the local repository was not configured to track a remote branch, run the command git branch --set-upstream-to=origin/master master to set the upstream branch.
  3. Try pushing my changes to the remote repository again using the git push command.
After following these steps, I was able to successfully push my changes to the remote repository on GitHub.
CSS Transition
I implemented a CSS transition on the .home_desc element that caused the font size to change when the element was hovered over. The transition was set to take 0.5 seconds to complete and used a cubic-bezier easing function to create a smooth animation. The final font size was set to 20px. I chose this CSS transition because I wanted to add a dynamic and visually appealing element to my website. The hover effect helped to draw the user's attention to the element and make the website more engaging. To make the transition work, I added the transition and :hover properties to the .home_desc element in my stylesheet.
